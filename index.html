ADDITIVE MANUFACTURING
FOR INNOVATIVE DESIGN

Additive Manufacturing (AM) is considered the future of rapid prototyping and product development. AM removes the limits of 3D designing posed by conventional subtractive approaches. AM is one of the most flexible manufacturing techniques and can work with PLA, ABS, TPU, PETg, and carbon fibre! AM exhibits flexibility through its applications, found in nanotechnology and building infrastructure on the surface of Mars. This course will provide you with an opportunity to design, develop and inspire. Here we focus massively on hands-on earning, paired with a variety of 3D printing technologies that provides you with an exciting flavour of 3D scanning and exposure to the word of reverse engineering. After this course, you will understand the concept of designing, rapid prototyping, product development, 3D scanning, and reverse engineering to meet the challenges of the future of manufacturing. Key technological aspects of this course:

  • FDM 3D Printers
  • SLA 3D Printers
  • 3D Scanners

MECHATRONICS BOOTCAMP:
A GATEWAY TO
EMERGING TECHNOLOGIES

This course is a combination of lab tasks, assessments, and projects related to the fundamental concepts of Mechatronics. It is curated for students with a curious minds, a keen interest in multidisciplinary engineering fields and wish to explore new technologies. This Bootcamp is all about getting familiar with the basics of programming, electrical circuit design, 3D modeling & prototyping. Therefore, we focus this emerging talent training program on developing these skills by employing hands-on experience and project-based earning.

  • Arduino Programming
  • Sensors and Actuators
  • Microcontrollers

EXPLORE THE FUTURE
WITH IOS APP DEVELOPMENT

The journey to becoming an iOS App developer begins with your imagination to create. If you are passionate about app development and programming, then this program is for you. Through the program, you will be able to develop the skill set to create and ideate. You will explore the world of iOS App development through Swift programming using Xcode IDE. This course will enable you to earn the art of using technological tools with a unique approach. By the end of this course, you will be ready to showcase your skills as a programmer and an iOS app developer. Key technological aspects of this course:

  • Swift Programming Language
  • XCode IDE
  • Software Design

FUNDAMENTALS OF
PROGRAMMING WITH
PYTHON

Python is one of the most demanded programming languages with applications found in building websites, data analysis, machine learning, natural language processing, data science, and artificial intelligence. It is designed for ease of use and powerful functionality, making it an idea language for beginners. This program will build on your programming foundation and will take your career to the next level. You will polish your skill set for interpreting complex data structures while developing improved algorithmic thinking and problem-solving capabilities. This course will practically teach you Python with interactive lectures, a full coding screencast and a corresponding code notebook! Learn in whatever manner is best for you! Key technological aspects of this course:

  • Pycharm, Jupyter Notebook
  • Google Colab
  • Automation

SCRATCH FOR GAME
DEVELOPERS

Scratch, developed by MIT, is the gold standard for learning to code creatively. This software helps develop computational thinking skills while removing barriers to self-expression through program design. This course demonstrates and explains how to write a great variety of programs in the Scratch programming language. Key technological aspects of this course:

  • Scratch IDE
  • Block Programming
  • Basic Game Development

GAME DEVELOPMENT
WITH UNREAL ENGINE

Learn game development with Unreal Engine 4. UE4 is the industry-leading 4D game design software that professionals use to create today’s top games. Unreal Engine has helped develop many blockbuster games and game franchises such as Borderlands, Gears of War, BioShock, Mass Effect, and several others. in this course, you will learn the fundamentals of game development using Unreal Engine 4 through a series of interactive sessions. By the end, you will have built an entire demo game from scratch. Upon completion, you will have the skills necessary to translate ideas into playable games that use environments you sculpt and logic that you define Key technological aspects of this course:

  • Extreme Graphics
  • Game Script and Logic
  • Professional Game Engine
